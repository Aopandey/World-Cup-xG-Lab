# Launch Checklist

## Build And Runtime

- [ ] Frontend typecheck passes.
- [ ] Frontend production build passes.
- [ ] Backend health endpoint passes.
- [ ] Docker stack runs locally.
- [ ] EC2 Docker stack runs.
- [ ] Deployment URL smoke test passes.
- [ ] HTTPS/domain is added before broad public sharing.

## Product QA

- [ ] Home page loads.
- [ ] Team pages load.
- [ ] Player pages load.
- [ ] Model page shows metrics.
- [ ] Coverage page explains source limits.
- [ ] Weak-sample warnings are visible.
- [ ] No fake player images are shown.
- [ ] No misleading prediction language is visible.
- [ ] Source badges explain StatsBomb, FBref, Understat, and DataMB.

## Documentation

- [ ] README is complete.
- [ ] Deployment docs are complete.
- [ ] Model card is complete.
- [ ] Data source documentation is complete.
- [ ] Portfolio case study is complete.
- [ ] Screenshots are added to `docs/screenshots/`.

## Demo Prep

- [ ] Prepare a 2-minute project walkthrough.
- [ ] Prepare one team example with strong historical data.
- [ ] Prepare one weak-sample player example.
- [ ] Prepare one model-metrics explanation.
- [ ] Prepare one data-limitation explanation.
